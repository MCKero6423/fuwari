# 泉盛UV-K6 V3刷机
  众所周知，新版的泉盛UV-K6使用的是新的mcu，这就导致原来的固件无法使用了。最近呢我购买了一台泉盛UV-K6，想给他刷机。但是呢刷入了旧版固件，开不了机。我就寻找到一个可以适配新版mcu的固件。

# 旧版的mcu
旧版的K6使用的是DP32G030这个mcu。这个mcu有着很高的可玩性。比如Egzumer，Losehu这些固件。都是基于这个DP32G030。这个mcu的生态也是极其的成熟。

# 新版的mcu
泉盛uvk6在2025年后半年开始使用新的mcu也就是py32。这就导致了以前的固件无法使用。

## 如果把旧mcu固件刷入新mcu的K6
我最近购买了一台K6，我在刷机的时候没有看，我就直接刷入了以前的固件，然后呢就导致开机不亮。但是还好，这影响不到再刷回去。但是刷都刷机了，我就想着折腾折腾吧。我就去互联网上寻找有没有适配新的mcu的固件。

# 适配新mcu的固件

F4HWN firmware port for the UV-K1 and UV-K5 V3 using the PY32F071 MCU
这款固件适配了，泉盛uvk5 v3和泉盛uvk6 v3（PY32F071 MCU）注意不要刷错了，不要V2刷了这个版本！新版本的K6还可以使用C口直接刷机不需要买写频线了（你也可以继续使用写频线）

请注意！
这个固件没有真正的大脑。请使用你自己的。使用此固件风险自负。完全不能保证它在你的无线电设备上能以任何方式正常工作，它甚至可能会把你的无线电设备弄坏，如果发生这种情况，你需要再买一个无线电设备。不管怎样，玩得开心。

这个固件做到了很多的功能，比如更好的s表，更多档位的功率（肯定没有超过5W的，就像调节音量，给你更多的细化选择，而非把音量调到超过硬件的性能）频谱分析仪，等等...
>更多请查看[固件文档](https://github.com/armel/uv-k1-k5v3-firmware-custom/wiki)

刷入这个固件，建议使用作者推荐的工具[UVTools2](https://armel.github.io/uvtools2/)不过请注意，刷机前请备份校准文件！一定要备份，一定要备份，一定要备份。

最新的v5.10还有一个截图功能，K5Viewe。这是作者在文档上说的教程。注意！K5Viewer 不是一个远程控制工具。

>这是我翻译的版本，并不100%准确，请查看[原文档](https://github.com/armel/uv-k1-k5v3-firmware-custom/wiki)

这是一个傻瓜式快速教程：

1.  为你的泉盛 K1 或 K5 刷入 F4HWN 固件
2.  保持对讲机通过 USB 线连接到你的电脑
3.  前往 [https://armel.github.io/k5viewer/](https://armel.github.io/k5viewer/)
4.  点击连接（并选择串口）

就是这样。无需安装。无需配置。即插即用 🤭
